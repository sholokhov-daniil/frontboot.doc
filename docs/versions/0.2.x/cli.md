# Консольные команды
Взаимодействие с модулем происходит через консоль, основанную на библиотеке [symfony/console](https://symfony.com/doc/current/components/console.html).  
Поддерживает работу через [bitrix cli](https://dev.1c-bitrix.ru/learning/course/index.php?COURSE_ID=43&LESSON_ID=11685).  
Обращение к консольным командам должно производиться в корневой директории модуля.

> [!WARNING]
> Все консольные команды должны выполняться с идентичным интерпретатором, который используется сайтом.
> 
> Версию php можно посмотреть по адресу: https://example.ru/bitrix/admin/phpinfo.php?test_var1=AAA&test_var2=BBB
> 
> Вместо **example.ru** укажите домен вашего сайта.

**Получение всех доступных версий php**
```bash
whereis php
```

**Выполнение консольной команды**

```bash
# Вместо {command} указываем реальное название команды
php ext {command}
```

**Справка по команде**

```bash
# Вместо {command} указываем реальное название команды
php ext {command} --help
```

## Список доступных команд

### Список доступных команд

```bash
php ext list
````

### Регистрация расширения

Регистрирует расширение в системе и позволяет хранить его в произвольном месте.

```bash
php ext reg

# или
php ext registration
```

После вызова команды нам потребуется указать идентификатор расширения и путь до корня расширения (без обязательного завершающего слэша)

### Генерация расширения

Автоматически создается зарегистрированная болванка вашего расширения

```bash
php ext create
```

Данная команда позволяет создавать следующие типы расширения:
- clean - Это чистое не модульное расширение, которое подходит для хранения простых js и css или ручной настройки
- vite-vue - Vue приложение от vite с конфигурацией по умолчанию

### Разрегистрация(удаление) расширения

Позволяет разрегистрировать существующее расширение и при необходимости может удалить все исходные файлы

```bash
# Вместо {extension} необходимо указать существующий ID расширения
php ext unregistration {extension}

# или
php ext unreg {extension}

# или
php ext del {extension}

# или
php ext delete {extension}
```

### Список зарегистрированных расширений

```bash
php ext extensions
```
